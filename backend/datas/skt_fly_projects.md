# [Project] 그리닷 (Greedot): 내가 그린 AI 단짝 친구

> **SKT FLY AI Challenger 4기 (Team 아보카도)**
> "내가 그린 그림이 갑자기 살아 움직인다?! 나랑 대화해줘!!"

---

## 1. 프로젝트 개요 (Overview)
* **서비스명:** 그리닷 (Greedot)
* **서비스 정의:** 손그림이 인격을 갖고 대화에 맞춰 움직이는 **메타 캐릭터 생성 서비스**
* **팀 정보:**
    * **팀명:** 아보카도
    * **팀원:** 변지협, 이성원, 김동욱, 김동재, 문지우
    * **멘토:** SKT 서정준

## 2. 기획 의도 및 목표 (Objectives)
이 서비스는 아동과 부모님 모두에게 필요한 가치를 제공합니다.

### 🎯 아동 (User)
* **상호작용:** 자신이 그린 캐릭터 '그리(Gree)'와의 대화를 통해 사회성을 발달시킵니다.
* **능력 향상:** 창의력과 상상력을 증진시킵니다.

### 🎯 부모님 (Guardian)
* **정서 케어:** 대화 로그 및 감정 분석 리포트를 통해 **자녀의 정서 모니터링**이 가능합니다.

---

## 3. 핵심 기능 및 특징 (Key Features)

1.  **높은 정확도의 감정 분석**
    * SKT의 **KoBERT** 모델을 활용하여 한국어 문맥에 특화된 정밀한 감정 분석을 수행합니다.
2.  **맞춤형 대화 (Personalized Chat)**
    * 사용자의 대화 내용을 분석하여 현재 감정 상태를 파악하고, 그에 맞는 위로, 공감, 놀이 등의 맞춤형 대화를 제공합니다.
3.  **생동감 있는 동작 (Lifelike Motion)**
    * **Meta Animated Drawings** 렌더링 기술을 적용하여, 정지된 손그림에 뼈대(Rigging)를 심어 걷거나 뛰는 등 생동감 있는 움직임을 부여합니다.

---

## 4. 시스템 아키텍처 (System Architecture)

### 🛠️ Tech Stack
* **Frontend:** Flutter (Cross-platform App)
* **Backend:** FastAPI (Python Web Framework)
* **Cloud Infrastructure (Azure):**
    * Azure VM (Virtual Machine)
    * Azure Blob Storage (이미지 데이터 저장)
    * Azure SQL Database (데이터베이스)
* **AI & API:**
    * **LLM:** OpenAI (ChatGPT API)
    * **Sentiment Analysis:** SKT KoBERT
    * **Vision/Motion:** Meta Animated Drawings

### 🔄 서비스 구동 프로세스 (Service Flow)
1.  **캐릭터 생성:** 아이가 스케치북에 그린 그림을 앱으로 촬영하여 업로드합니다.
2.  **AI 변환 & 리깅:** AI가 그림의 윤곽선을 따고 관절을 인식하여 움직이는 캐릭터로 변환합니다.
3.  **대화 하기:** 캐릭터가 인사를 건네며("안녕? 나는 너만의 친구야!"), 아이와 음성/텍스트로 대화합니다.
4.  **감정 분석:** 대화 내용은 실시간으로 분석되어 아이의 기분 상태를 파악합니다.
5.  **리포트 발행:** 부모님 앱에서 아이의 대화 키워드, 감정 변화 추이, 관심사 등을 시각화된 그래프로 확인합니다.
